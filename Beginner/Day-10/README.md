# 🧮 Day 10 - Calculator

A command-line calculator built in Python as part of **Day 10** of the **100 Days of Code: Python Bootcamp**. The program performs basic arithmetic operations using functions and a dictionary to map operators to their corresponding functions.

---

## 📂 Project Structure

```text
Day-10/
│── main.py
│── art.py
└── README.md
```

---

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- Continue calculations using the previous result
- Restart the calculator anytime
- Function-based implementation
- Dictionary mapping for operations
- ASCII logo

---

## 🛠️ Concepts Used

- Functions
- Dictionaries
- Loops
- Conditional Statements
- Function as Values
- User Input
- Modular Programming
