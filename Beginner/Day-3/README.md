# 🏝️ Day 3 - Treasure Island

## 📌 Overview

**Treasure Island** is a text-based adventure game where players make choices that determine the outcome of the story.

This project introduces decision-making in Python using conditional statements and logical operators.

---

## 🚀 Concepts Learned

- if statements
- if-else conditions
- elif statements
- Nested conditions
- Logical operators
- Control flow

---

## ✨ Features

- Interactive storyline
- Multiple decision paths
- Different endings
- User-driven gameplay

---

## 🖥️ Sample Output

```text
Welcome to Treasure Island.

You're at a crossroad.
Where do you want to go?

Type "left" or "right"
> left

You've come to a lake.
Type "wait" to wait for a boat.
Type "swim" to swim across.
> wait

There are three doors.
One red, one yellow and one blue.
Which colour do you choose?
> yellow

🎉 You found the treasure!
You Win!
```
