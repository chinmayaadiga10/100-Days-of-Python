# 💰 Day 2 - Tip Calculator

## 📌 Overview

The **Tip Calculator** calculates how much each person should pay when splitting a restaurant bill, including a chosen tip percentage.

This project focuses on numerical data types, mathematical operations, type conversion, and formatted output.

---

## 🚀 Concepts Learned

- Integer, float, and string data types
- Type conversion
- Mathematical operators
- Floating-point calculations
- f-strings
- Rounding numbers

---

## ✨ Features

- Accepts total bill amount
- Allows users to choose a tip percentage
- Splits the bill among multiple people
- Displays the amount each person should pay

---

## 🖥️ Sample Output

```text
Welcome to the Tip Calculator!

What was the total bill? ₹1500
How much tip would you like to give? 10
How many people to split the bill? 3

Each person should pay: ₹550.00
```
