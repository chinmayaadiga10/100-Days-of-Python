# 🎯 Day 7 - Hangman Game

A simple command-line implementation of the classic **Hangman** game built in Python as part of **Day 7** of Dr. Angela Yu's **100 Days of Code: Python Bootcamp**.

---

## 📖 Overview

The computer randomly selects a word, and the player guesses one letter at a time before running out of lives. The project combines many Python fundamentals learned during the first week of the course into a complete interactive game.

---

## 🚀 Features

- 🎲 Random word selection
- 🔤 Letter-by-letter guessing
- ❤️ Limited lives system
- 🎨 ASCII Hangman graphics
- ✅ Win and Lose conditions
- 📦 Modular code using separate Python files

---

## 🛠️ Technologies Used

- Python 3
- `random` module
