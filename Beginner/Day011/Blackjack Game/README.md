# 🃏 Day 11 - Blackjack Game

A command-line implementation of the classic **Blackjack** card game built in Python as part of **Day 11** of the **100 Days of Code: Python Bootcamp**.

---

## 🚀 Features

- Random card dealing
- Blackjack detection
- Ace value adjustment (11 → 1)
- Player can hit or stand
- Computer follows Blackjack rules (draws until 17)
- Win, lose, and draw detection
- Play multiple rounds
- ASCII logo

---

## 🛠️ Concepts Used

- Functions
- Lists
- Loops
- Conditional Statements
- Random Module
- Return Values
- Game Logic
- Modular Programming

---

## ▶️ Run Locally

```bash
git clone https://github.com/chinmayaadiga10/100-Days-of-Python.git

cd Beginner/Day011

python main.py
```
