# 💰 Day 9 - Secret Auction

A command-line Secret Auction program built in Python as part of **Day 9** of the **100 Days of Code: Python Bootcamp**. The program collects bids from multiple users and determines the highest bidder using Python dictionaries. :contentReference[oaicite:0]{index=0}

## 🚀 Features

- Accepts bids from multiple users
- Stores bids using a Python dictionary
- Clears the screen between bidders
- Finds and displays the highest bidder
- ASCII logo
