# 🎯 Day 12 - Number Guessing Game

A command-line Number Guessing Game built in Python as part of **Day 12** of the **100 Days of Code: Python Bootcamp**. The player has to guess a randomly generated number between **1 and 100** within a limited number of attempts.

---

## 🚀 Features

- Random number generation
- Easy (10 attempts) and Hard (5 attempts) difficulty modes
- Feedback for high and low guesses
- Win and lose conditions
- ASCII logo

---

## 🛠️ Concepts Used

- Functions
- Loops
- Conditional Statements
- Random Module
- User Input
- Variables

---

## ▶️ Run Locally

```bash
git clone https://github.com/chinmayaadiga10/100-Days-of-Python.git

cd Beginner/day-012

python main.py
```
