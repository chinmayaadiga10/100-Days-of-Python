# 🎸 Day 1 - Band Name Generator

## 📌 Overview

The **Band Name Generator** is the first project in the **100 Days of Python** course by Dr. Angela Yu.

This project introduces fundamental Python concepts such as user input, variables, string concatenation, and output formatting. The program generates a potential band name using information provided by the user.

---

## 🚀 Concepts Learned

- Printing to the console
- Python variables
- User input using `input()`
- String manipulation
- Variable naming conventions
- Combining strings

---

## ✨ Features

- Takes the user's city name
- Takes the name of their pet
- Generates a band name suggestion
- Beginner-friendly implementation

---

## 🖥️ Sample Output

```text
Welcome to the Band Name Generator!

What's the name of the city you grew up in?
Bangalore

What's your pet's name?
Bruno

Your band name could be:
Bangalore Bruno
```
