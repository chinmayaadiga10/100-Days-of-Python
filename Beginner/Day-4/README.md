# 🪨 Day 4 - Rock Paper Scissors

## 📌 Overview

The **Rock Paper Scissors** project is a simple Python game where the user competes against the computer.

The player chooses Rock, Paper, or Scissors, and the computer makes a random choice. The program then determines the winner based on the game rules.

---

## 🚀 Concepts Learned

- Randomisation using the `random` module
- Python Lists
- List indexing
- Conditional statements (`if`, `elif`, `else`)
- Nested conditions
- User input and output

---

## ✨ Features

- User chooses Rock, Paper, or Scissors
- Computer generates a random choice
- Displays both choices using ASCII art
- Determines whether the user wins, loses, or draws

---
