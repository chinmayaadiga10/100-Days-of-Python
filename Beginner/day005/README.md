# 🔐 Day 5 - Password Generator

## 📌 Overview

The **Password Generator** project creates a random password based on the user's preferences.

The user specifies the number of letters, symbols, and numbers they want in their password, and the program generates a secure random password using those inputs.

---

## 🚀 Concepts Learned

- Python `for` loops
- The `range()` function
- Randomization using the `random` module
- Lists and list operations
- String concatenation
- Looping through sequences

---

## ✨ Features

- User chooses the number of letters
- User chooses the number of symbols
- User chooses the number of numbers
- Generates a random password
- Shuffles characters for increased randomness

---

## ▶️ How to Run

1. Open the project folder.
2. Run:

```bash
python main.py
```

3. Enter the required number of letters, symbols, and numbers.
4. The program will generate a random password.
