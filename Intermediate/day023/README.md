# 🐢 Day 23 - Turtle Crossing Game

A Frogger-inspired arcade game built in Python using the `turtle` module as part of **Day 23** of the **100 Days of Code: Python Bootcamp**. The objective is to safely guide the turtle across the road while avoiding moving cars. :contentReference[oaicite:0]{index=0}

## 🚀 Features

- Player-controlled turtle
- Randomly generated moving cars
- Collision detection
- Level progression
- Increasing car speed with each level
- Game over screen
