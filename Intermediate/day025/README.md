# 🗺️ Day 25 - U.S. States Game

An interactive geography game built in Python using the **Turtle Graphics** and **Pandas** libraries as part of **Day 25** of the **100 Days of Code: Python Bootcamp**. The player guesses the names of all 50 U.S. states, which are displayed on the map as they are correctly identified.

---

## 📂 Project Structure

```text
day025/
│── main.py
│── 50_states.csv
│── blank_states_img.gif
│── states_to_learn.csv
└── README.md
```

---

## 🚀 Features

- Interactive U.S. map using Turtle Graphics
- Guess all 50 U.S. states
- Displays correctly guessed states on the map
- Prevents duplicate guesses
- Generates a `states_to_learn.csv` file containing the states that were missed
- Uses CSV data to position state names accurately
