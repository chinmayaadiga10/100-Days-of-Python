# 🎨 Day 18 - Hirst Painting

A Python program built using the `turtle` module that recreates a **Hirst-style dot painting**. The colors are extracted from an image using the **colorgram** library and used to generate a grid of colorful dots.

---

## 📂 Project Structure

```text
Day-18/
│── main.py
│── image.jpg
└── README.md
```

---

## 🚀 Features

- Extracts colors from an image using `colorgram`
- Generates a 10 × 10 grid of colored dots
- Randomly selects colors for each dot
- Uses the `turtle` module for drawing
- RGB color mode support

---

## 🛠️ Concepts Used

- Turtle Graphics
- External Python Packages
- Lists
- Tuples
- Loops
- Random Module
- Functions

---

## ▶️ Run Locally

Install the required package:

```bash
pip install colorgram.py
```

Run the program:

```bash
python main.py
```
