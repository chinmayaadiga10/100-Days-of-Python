# ✈️ Day 26 - NATO Phonetic Alphabet

A Python program that converts a user-entered word into its **NATO Phonetic Alphabet** equivalent using **Pandas**, **dictionary comprehension**, and **file handling**. This project is part of **Day 26** of the **100 Days of Code: Python Bootcamp**.

---

## 📂 Project Structure

```text
day026/
│── main.py
│── nato_phonetic_alphabet.csv
└── README.md
```

---

## 🚀 Features

- Reads the NATO alphabet from a CSV file
- Creates a dictionary using dictionary comprehension
- Converts any word into its NATO phonetic representation
- Accepts user input from the terminal
- Uses Pandas for data processing
