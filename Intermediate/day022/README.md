# 🏓 Day 22 - Pong Game

A recreation of the classic **Pong** game built in Python using the `turtle` module as part of **Day 22** of the **100 Days of Code: Python Bootcamp**.

---

## 📂 Project Structure

```text
Day-22/
│── main.py
│── paddle.py
│── ball.py
│── scoreboard.py
└── README.md
```

---

## 🚀 Features

- Two-player Pong game
- Keyboard-controlled paddles
- Ball movement and collision detection
- Paddle and wall collisions
- Automatic ball reset after scoring
- Live score tracking
- Increasing ball speed during gameplay

---

## 🛠️ Concepts Used

- Object-Oriented Programming (OOP)
- Classes and Objects
- Turtle Graphics
- Inheritance
- Collision Detection
- Event Listeners
- Loops
- Modular Programming
