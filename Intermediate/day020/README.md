# 🐍 Day 20 - Snake Game (Part 1)

A Python implementation of the classic **Snake Game** built using the `turtle` module as part of **Day 20** of the **100 Days of Code: Python Bootcamp**. This project focuses on creating the snake, movement mechanics, and screen animation..

---

## 📂 Project Structure

```text
.
├── day020/
│   ├── data.txt
│   ├── Food.py
│   ├── scoreboard.py
│   ├── snake.py
│   └── main.py
└── README.md
```

## 🚀 Features

- Snake created using multiple turtle segments
- Smooth movement animation
- Directional controls using arrow keys
- Snake movement without reversing direction
- Object-oriented implementation
