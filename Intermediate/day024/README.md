# 📧 Day 24 - Mail Merge

A Python automation project that generates personalized invitation letters using a template and a list of names. This project focuses on **file handling**, **string manipulation**, and **automating repetitive tasks** using Python.

---

## 📂 Project Structure

```text
day024/
│── Input/
│   ├── Letters/
│   │   └── starting_letter.txt
│   └── Names/
│       └── invited_names.txt
│
│── Output/
│   └── ReadyToSend/
│
├── main.py
└── README.md
```

---

## 🚀 Features

- Reads a list of names from a text file
- Reads a letter template
- Replaces the placeholder (`[name]`) with each recipient's name
- Generates a personalized letter for every recipient
- Saves all generated letters in the output folder

---

## 🛠️ Concepts Used

- File Handling (`open`, `read`, `write`)
- Context Managers (`with`)
- Relative File Paths
- String Manipulation
- `replace()` Method
- Loops
